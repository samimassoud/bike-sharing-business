# Bike Share Analysis
## Project Overview
This project analyzes bike share data from Chicago, New York, and
Washington to derive insights into the usage patterns.
## Datasets
- Chicago: `chicago.csv`
- New York: `new_york_city.csv`
- Washington: `washington.csv`
